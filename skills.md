# Agent Skills: Livestock & Farm Management System

## Core Competencies
* **UI/UX Design:** Ability to replicate a Material Design dashboard with a bottom navigation bar (Nyumbani, Tafuta, Daktari, Mimi) and a central Floating Action Button (+).
* **Localization:** Proficiency in Swahili for all UI elements (e.g., "Habari ya mchana," "Rekodi za Flock").
* **Data Modeling:** Capability to structure a relational database (Firebase Firestore) for the following collections:
    * **Users:** Roles for Farmers (Mkulima) and Service Providers (Daktari).
    * **Flock:** Tracking "Kuku" (Poultry) counts and health status.
    * **Marketplace:** Managing "Bidhaa Zangu" (My Products) for the "Soko" (Market).
    * **Consultations:** Scheduling "Dharura" (Emergency) and routine vet visits.

## Functional Workflows
1. **Dashboard Aggregation:** Calculate and display real-time counts for Miamala (Transactions), Maoni (Feedback), and Ratings.
2. **Search & Filter:** Implement a search bar for finding "Madaktari wa Mifugo" with status filters (Wanapatikana sasa).
3. **Record Management:** Create forms to input and display "HealthRecord" and "SuccessStory" entries.
4. **State Management:** Maintain the app state across the Preview and Dashboard views as seen in the development environment.

## Integration Skills
* **Analytics:** Connect to Google/Firebase Analytics to track "Live visitors" and user engagement.
* **Notifications:** Setup a push notification system for the "Dharura" (Emergency) alert system.
